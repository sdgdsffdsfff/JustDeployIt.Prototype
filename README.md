# JustDeployIt功能原型页面 #

> 基于[Blue Moon][1]Bootstrap模板

使用了自制的简单页面拼接工具，便于进行原子化的页面组装，提高效果预览的效率，降低修改成本。

## 页面元素应用规则 ##

**Button**
- btn-primary
 - 创建项目
 - 创建部署
- btn-success
 - 执行...
- btn-info
 - 添加服务器
 - 添加服务器组
 - 添加...
 - 预览
- btn-danger
 - 删除
- btn-warning
 - 修改

**Panel**

- panel-info
 - 展示信息 



[1]: http://iamsrinu.com/bluemoon-admin-theme4/
